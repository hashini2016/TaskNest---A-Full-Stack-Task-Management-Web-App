# Project Setup Instructions

This project contains a *frontend* and a *backend*. Follow the instructions below to set up the project locally.

## Prerequisites

Make sure you have the following installed:

* Node.js (v16 or higher recommended)
* npm (comes with Node.js)

## Frontend Setup

1. Open a terminal and navigate to the frontend folder:
bash
cd frontend

2. Install the dependencies:
bash
npm install --force


3. Start the frontend server (if applicable):
bash
npm start

## Backend Setup

1. Open a terminal and navigate to the backend folder:
bash
cd backend

2. Install the dependencies:
bash
npm install

3. Start the backend server (if applicable):
bash
npm start

## Notes

* Make sure both frontend and backend servers are running for the application to work properly.
* If you encounter dependency issues, try removing node_modules and reinstalling:
